# QueueFest

An Event Ordering App UI Flow built with React, Vite, and Tailwind CSS. The original design is available on [Figma](https://www.figma.com/design/nyQodf71c9nypQ7nl9609X/Event-Ordering-App-UI-Flow).

## Prerequisites

Before getting started, make sure you have the following tools installed on your system:

- [Node.js](https://nodejs.org/) (v18 or higher)
- [pnpm](https://pnpm.io/) (recommended package manager)

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/dlal2-ua/QueueFest.git
   cd QueueFest
   ```

2. Install the project dependencies:

   ```bash
   pnpm install
   ```

## Usage

### Development Server

To launch the local development server with hot-reload:

```bash
pnpm run dev
```

The application will be available at the URL displayed in the terminal (by default, `http://localhost:5173`).

### Production Build

To generate an optimized production build:

```bash
pnpm run build
```

The compiled output will be placed in the `dist/` directory.
